# idtfrontend

Front-end documentation, built with [DocPad](http://docpad.org).


## Dependencies

* [node.js](http://nodejs.org/)
* [npm](https://npmjs.org)


## First run

Clone the repository.

``` bash
$ npm install
$ bower install
$ grunt
```
Open http://localhost:7777/docs/
